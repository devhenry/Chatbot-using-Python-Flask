# Chatbot-using-Python-Flask
This is a secure, local-first AI Chatbot designed for document interrogation, allowing users to chat directly with the content of their PDF files completely offline.

Tired of uploading sensitive documents to external APIs? This project presents a powerful, 100% offline AI chatbot built to read, understand, and answer questions directly from your PDF files.

Developed using Python and the LangChain framework, this system uses a local LLM (GPT4All) and FAISS for vector search, ensuring absolute privacy and eliminating recurring API costs. Just upload your PDF and start querying.

Core Capabilities:

Privacy First: All data processing and inference are performed locally.

Contextual Q&A: Get accurate, sourced answers from your uploaded PDF content.

Seamless Interface: Features include PDF preview, real-time response generation, and easy chat history management (built with Flask and responsive web technologies).

Tech Stack Snapshot: Python 3.10+, Flask, LangChain, GPT4All, and FAISS.
